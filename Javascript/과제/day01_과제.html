콘솔창 사칙연산 계산기

이 계산기는 웹 브라우저 콘솔창에서 실행하는 프로그램입니다. 콘솔창을 연 뒤 start()를 입력하면 계산기를 사용할 수 있습니다.

Windows OS

  1. F12 키를 눌러 개발자 도구를 엽니다.
  2. 상단 메뉴에서 Console 탭을 선택합니다.
  3. 콘솔창에 start()를 입력하고 Enter를 누릅니다.

Mac OS

  1. Option + Command + I 키를 눌러 개발자 도구를 엽니다.
  2. 상단 메뉴에서 Console 탭을 선택합니다.
  3. 콘솔창에 start()를 입력하고 Enter를 누릅니다.
