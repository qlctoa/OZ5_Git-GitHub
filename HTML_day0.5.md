HTML; HyperText Markup Language
: 웹 페이지의 구조를 정의
- 글의 제복, 문단, 이미지, 링크 같은 웹페이지의 뼈대를 만든 역할을 함
- `<tag>`들의 조합으로 구성
    - `<태그 이름>` 형태로 작성
    - 각 태그마다 요소의 의미와 역할을 가짐
    - 여는 태그 `<tag>`, 닫는 태그 `</tag>`
---
예시
```html
<!DOCTYPE html>
<html>
    <head>
        <title>MY PAGE</title>
    </head>
    <body>
        hello
    </body>
</html>
```
